bling's vim distribution

Here you'll find information about my vim distribution. I recommend that you not use this verbatim, but rather, take ideas from it to build your own vimrc.

However, it does work out of the box with minimal configuration. Just following these steps to make it happen!

  1. git clone https://github.com/bling/dotvim ~/.vim
  2. cd ~/.vim
  3. git submodule init && git submodule update
  4. ln -s ~/.vim/vimrc ~/.vimrc (don't forget to make a backup of your existing vimrc)
